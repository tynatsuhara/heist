# Tiny Heist
A voxel-based destruction gorefest where you have one objective - cold, hard cash.

## Controls
* E - Interact
* F - Draw Weapon, Shout
* G - Throw bag
* SPACE - Drag body
* R - Reload
* Mouse0 - Shoot
* Mouse1 - Melee
* MouseWheel - Zoom
* ESC - Pause
* N on pause menu - Restart level
